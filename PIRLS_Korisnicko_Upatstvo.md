## 1. Вовед

**PIRLS Дигитален Тренер** е бесплатна веб-апликација за подготовка на учениците и наставниците за PIRLS — меѓународното истражување за читалечка писменост. Апликацијата содржи **15 приказни** (наративни и информативни текстови), секоја придружена со прашања, илустрации и педагошки алатки.

### За кого е наменета?

| Корисник | Употреба |
|----------|----------|
| **Ученик (4. одд.)** | Самостојно читање и одговарање на прашања |
| **Наставник** | Следење во реално време, доделување текстови, преглед на одговори |
| **Родител** | Поддршка на дома — детето работи самостојно |

### Технички барања

- Модерен прелистувач: Chrome, Firefox, Edge, Safari
- Работи на: компјутер, таблет, мобилен телефон
- Интернет врска (препорачана); основните функции работат и офлајн
- Нема инсталација — само отворете го линкот

## 3. Упатство за ученици

### 3.1 Два начина на работа

**А. Самостојно (без наставник)**

1. Отворете ја апликацијата
2. Директно кликнете на некоја приказна
3. Читајте и одговарајте самостојно
4. Одговорите се зачувуваат само на вашиот уред

**Б. Во час (со наставник)**

1. Наставникот ви дава **6-значен код** (пример: `KFX7PQ`)
2. Кликнете **„Влези во час"** (горе лево)
3. Внесете го вашето **ime и prezime** и **кодот**
4. Кликнете **„Влези"**

> ✅ Ако влезовте успешно, горе лево ќе видите зелена значка со вашето ime и кодот на часот.

---

### 3.2 Влез во час — чекор по чекор

```
[ Влези во час ]
       ↓
┌─────────────────────────────┐
│  Твоето ime и prezime:      │
│  [________________________] │
│                             │
│  Код на часот:              │
│  [________________________] │
│                             │
│  [ Влези ]   [ Прескокни ]  │
└─────────────────────────────┘
```

**Важно:**
- Кодот го дава наставникот (6 букви/бројки, не е чувствителен на мали/големи букви)
- Ако кликнете **„Прескокни"**, ќе работите самостојно — наставникот нема да ги гледа вашите одговори
- Ако кодот не постои, ќе добиете порака: *„Кодот не постои. Прашај го наставникот."*

---

### 3.3 Читање на приказна

По изборот на приказна се отвора **екранот за читање**:

```
┌─────────────────────────────────────────────────────┐
│  ← Назад    🧩 Слагалка  🎨 Боенка    ⏱️ 2:34       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  [Илустрација]                                      │
│                                                     │
│  Текстот на приказната...                           │
│  Клучните зборови се подвлечени.                    │
│                                                     │
│  [Уште илустрации вметнати во текстот]              │
│                                                     │
│  ...продолжува текстот...                           │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [A+] [A-]  [🖍️ Обележувај]  [ 📋 Прашања ]         │
└─────────────────────────────────────────────────────┘
```

**Трака горе:**
- `← Назад` — враќа на почетниот екран
- `🧩 Слагалка` — хронолошка слагалка (ако е достапна)
- `🎨 Боенка` — дигитална боенка (ако е достапна)
- `⏱️` — тajмер, го покажува поминатото време

**Трака долу:**
- `A+` / `A-` — зголемување/намалување на фонтот
- `🖍️ Обележувај` — активира режим за обележување текст
- `📋 Прашања` — ги отвора прашањата

---

### 3.4 Интерактивен речник

Додека читате, ќе забележите **подвлечени зборови** во текстот (во виолетова боја). Тоа се клучни поими со дефиниции.

**Како да го користите:**
1. Кликнете на подвлечениот збор
2. Ќе се отвори мал прозорец со дефиниција
3. Кликнете надвор за да го затворите

**Пример:**
> „Октоподот има *пипала*..."
> → Кликнете на **пипала** → *„Долги, флексибилни органи кај некои животни кои служат за фаќање, движење или допир."*

### 3.5 Одговарање на прашања

Кликнете **„📋 Прашања"** за да ги отворите прашањата. Тие се подигнуваат од дното на екранот.

```
┌─────────────────────────────────────────────────────┐
│  ПРАШАЊЕ 1 / 12                    ████░░░░ 33%     │
│─────────────────────────────────────────────────────│
│                                                     │
│  Зошто Марко требаше да го помене распустот         │
│  во селото?                                         │
│                                                     │
│  [ А) Затоа што сакаше да истражува.             ]  │
│  [ Б) Затоа што неговите родители патуваа.  ✅  ]  │
│  [ В) Затоа што сакаше богатство.                ]  │
│  [ Г) Затоа што му беше досадно.                 ]  │
│                                                     │
│  🦉 „ТОЧНО! Одличен одговор!"                       │
│─────────────────────────────────────────────────────│
│            [← ] Назад   [ →] Следно                │
└─────────────────────────────────────────────────────┘
```

#### Прашања со избор (MCQ)

- Имате 4 можности
- Кликнете на вашиот избор
- Ако е **точно** → ✅ и пораката на совичката, автоматски продолжувате
- Ако е **погрешно** → порака: *„Хм... Одговорот е во текстот! Обиди се пак."*

#### Прашања со отворен одговор

```
┌─────────────────────────────────────────────────────┐
│  Как се чувствуваше Марко кога го отвори            │
│  ковчегот? Објасни зошто.                           │
│                                                     │
│  ┌───────────────────────────────────────────────┐  │
│  │ Марко беше разочаран бидејќи очекуваше        │  │
│  │ да најде злато, но внатре имаше само           │  │
│  │ обични предмети.                               │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
│  [ 💡 Совет ]              [ ✅ Потврди ]           │
└─────────────────────────────────────────────────────┘
```

- Напишете го вашиот одговор во полето
- Потребни се **најмалку 6 знаци**
- Кликнете **„✅ Потврди"**

**Нивоа на повратна информација:**

| Повратна информација | Значење |
|----------------------|---------|
| 🏆 **ИЗВОНРЕДНО!** | Одговорот содржи клучни детали — одличен! |
| ✨ **Добар почеток!** | Споменавте еден детаљ, додајте уште еден |
| 🤔 **Обиди се пак** | Одговорот не содржи клучни информации |

> 💡 **Совет:** Ако не знаете како да одговорите, кликнете **„💡 Совет"** за да добиете насока без готов одговор.

На почетниот екран, завршените приказни се означени со 🎖️. Ако наставникот доделил одредена приказна, таа е означена со 📌 **„Денес"** и другите се заклучени.

## 5. Алатки при читање

### 5.1 Промена на големина на фонтот

Во долната трака: **`A+`** (поголем) и **`A-`** (помал)

- Минимум: мал фонт
- Максимум: многу голем фонт
- Корисно за ученици со слабовид

### 5.2 Обележување текст (хајлајтинг)

1. Кликнете **„🖍️ Обележувај"**
2. Изберете боја: 🟡 Жолта / 🟢 Зелена / 🔵 Сина
3. Изберете текст со глувчето (или прст на таблет)
4. Текстот се обележува со избраната боја
5. За бришење: кликнете на веќе обележаниот дел
6. За бришење на сите обележувања: **„Исчисти сè"**

> **Педагошки совет:** Учениците можат да ги обележуваат клучните информации пред да одговараат на прашањата — директна стратегија за читање со разбирање.

### 5.3 Тajмер

Тajмерот автоматски почнува кога се отвора приказната. Ги покажува минутите и секундите.

Корисно за: практикување работа под временски притисок (симулација на реален PIRLS тест).

### 5.4 Хронолошка слагалка 🧩

*(Достапна за избрани приказни)*

- Кликнете **„🧩 Слагалка"** во горната трака
- Потребно е да ги подредите сцените од приказната по правилен редослед
- Кликнете на сцена → кликнете на местото каде сака да оди

### 5.5 Дигитална боенка 🎨

*(Достапна за избрани приказни)*

- Кликнете **„🎨 Боенка"** во горната трака
- Изберете боја и боjте ги илустрациите

## 6. Прашања и одговори

### 6.1 Навигација низ прашањата

```
[ ← ] Претходно    Прашање 3 / 12    [ → ] Следно
```

- Со стрелките се движите напред и назад
- Прашањата нема задолжителен редослед — можете да се вратите назад
- Прогресбарот покажува % завршени прашања

### 6.2 Совет (💡 Hint)

За секое отворено прашање постои необврзувачки совет:
- Кликнете **„💡 Совет"**
- Добивате насока за размислување (не готов одговор)
- Пример: *„Погледни го третиот параграф. Кои зборови го покажуваат чувството на Марко?"*

### 6.3 Читање на глас 🔊

*(Доколку вашиот уред поддржува македонски глас)*

- До секое прашање постои копче **🔊**
- Кликнете за да го слушнете прашањето прочитано на глас
- Корисно за ученици со потешкотии при читање

## 7. Достапност и прилагодувања за ученици со посебни образовни потреби (ПОП)

> **За наставниците:** Овој дел е напишан и за вас. За секоја категорија ПОП се дадени конкретни чекори за поставување на апликацијата **пред часот**, препораки за работа **за време на часот** и насоки за **следење на прогресот** преку таблата.

---

### 7.1 За ученици со дислексија

Учениците со дислексија имаат потешкотии со декодирање на текст, па вложуваат повеќе когнитивна енергија во читањето и им останува помалку за разбирање. Клучно е да се намали напорот при декодирање и да се обезбеди мултисензорна поддршка.

#### Поставување пред часот — дислексија

1. **Зголемете го фонтот** пред да им го дадете уредот: кликнете `A+` повеќе пати до удобна големина
2. **Активирајте ги советите однапред**: информирајте го ученикот дека може слободно да го користи 💡 Совет без да се чувствува „слаб"
3. **Изберете наративна приказна** за прв обид (пр. *Тајната на ковчегот* — линеарна нарација, многу илустрации)

#### Алатки за ученици со дислексија

| Алатка | Зошто помага кај дислексија |
| ------ | --------------------------- |
| `A+` фонт | Помал број букви по ред → помалку визуелно „збивање" |
| Илустрации во текстот | Даваат контекст пред и за време на читањето — намалуваат потребата за повторно читање |
| Интерактивен речник | Значењето на тешкиот збор се добива со еден клик, без напуштање на текстот |
| 🔊 Читање на глас | Аудитивна поддршка — ученикот слуша додека чита — двоканален влез |
| 💡 Совет | Ја активира работната меморија без притисок на јазична продукција |
| Повторено читање | Нема ограничување — ученикот го прочитува текстот колку пати е потребно |
| Обележување со 🖍️ | Ученикот ги маркира клучните реченици пред прашањата — активна стратегија за читање |

#### Чекори за час — дислексија

```
Чекор 1: Наставникот го зголемува фонтот на уредот на ученикот (A+ × 3–4)
Чекор 2: Ученикот ја чита приказната ЕДНАШ без прашања — само за уживање
Чекор 3: Ученикот ја прочитува втор пат, со 🖍️ обележување на клучни делови
Чекор 4: Ученикот ги отвора прашањата — прво MCQ, потоа отворени
Чекор 5: За секое отворено прашање — наставникот му нуди 💡 Совет ако затежне
```

#### Следење преку таблата — дислексија

- Напредокот е побавен → нормално, не прекинувајте
- Ако ученикот стагнира на исто прашање → активирајте го Советот
- Погрешните MCQ одговори кај дислексија честопати се должат на **брзо читање на прашањето**, не на непознавање — водете дискусија по часот

### 7.2 За ученици со АДХД

Учениците со АДХД имаат потешкотии со одржување на вниманието на подолг текст, регулација на импулсивноста при одговарањето и работа во услови кои нудат многу стимули. Потребна е јасна структура, кратки задачи и непосредна награда.

#### Поставување пред часот — АДХД

1. **Исклучете го тajмерот** ако притискот на времето предизвикува анксиозност (тajмерот само брои, не ограничува — но видот на течечките бројки може да дистрактира)
2. **Доделете само еден текст** — заклучете ги другите приказни со функцијата „Задај приказна" за да нема визуелни дистрактори
3. **Договорете мини-цели**: „Прочитај само до првата слика, па застани"

#### Алатки за ученици со АДХД

| Алатка | Зошто помага кај АДХД |
| ------ | --------------------- |
| Тajмер (само бројач) | Ако не дистрактира, му помага на ученикот да ги следи поминатите минути |
| Едно прашање на екран | Нема преоптоварување — само еден фокус |
| Непосредна повратна информација | Звук и порака веднаш → задоволство и мотивација за следното |
| Медалка 🎖️ по приказна | Конкретна, видлива награда → позитивно засилување |
| Прогресбар | Визуелен напредок (`████░░ 67%`) → го одржува мотивот |
| MCQ пред отворени | Структурирани избори прво → се намалува анксиозноста |
| 💡 Совет | Ако ученикот се „изгуби" во размислување, насоката го враќа на задачата |

#### Чекори за час — АДХД

```
Чекор 1: Договорете правило: „Прво читаме, потоа прашања — нема скокање"
Чекор 2: Читање во сегменти — читај до илустрацијата, застани, разговарај
Чекор 3: Прашањата — наставникот ги отвора заедно со ученикот (не самостојно)
Чекор 4: По секое точно одговорено прашање — мал пауза, устана движба (ако е потребна)
Чекор 5: MCQ прашањата прво, отворените на крај или следниот час
```

#### Следење преку таблата — АДХД

- Ако ученикот „скока" меѓу приказни → доделете му само еден текст
- Ако напредокот е 0% по 10 минути → проверете дали е на прашањата или сè уште чита
- Многу брзи, нерефлексивни MCQ одговори → пред часот договорете „прочитај двапати пред да кликнеш"

### 7.3 За ученици со потешкотии во читање (без дијагноза)

Дел од учениците немаат формална дијагноза, но читаат под нивото на одделението — побавно, со помало разбирање, со ограничен речник. Ова е честа ситуација во хетерогени одделенија.

#### Алатки за слаби читачи

| Алатка | Зошто помага |
| ------ | ------------ |
| Интерактивен речник | Речничкиот дефицит е директно надополнет — без прекинување на читањето |
| Илустрации | Градат „менталната слика" пред читање на параграфот |
| 🔊 Читање на глас | Аудитивниот влез го поддржува визуелното декодирање |
| Обележување | Ги идентификуваат клучните информации пред да одговараат |
| Повратна информација со клучни поими | „Добар почеток! Ти кажа `разочаран`, додај уште еден детал" — покажува кои зборови се важни |

#### Диференцирана употреба во мешано одделение

Апликацијата му овозможува на наставникот да работи со целото одделение на **ист текст**, но со различно темпо и поддршка:

```
Напредни ученици:      Читаат самостојно → одговараат без Совет
Просечни ученици:      Читаат самостојно → Совет при потреба
Ученици со потешкотии: Читаат со 🔊 → Совет активен → MCQ прво
```

Наставникот ги следи сите три групи **истовремено** преку таблата.

| ------ | ----- |

2. Потоа дополнително користете `A+` во апликацијата
3. Активирајте го `🔊` за прашањата

### 7.5 За ученици со јазични тешкотии / слаба јазична компетентност

Вклучувајќи ученици кои македонскиот го учат како втор јазик. Овие ученици разбираат одредена порака, но имаат потешкотии да ги вербализираат сознанијата — особено при отворените прашања каде треба да произведат текст.

#### Алатки за јазични тешкотии

| Алатка | Зошто помага |
| ------ | ------------ |
| Интерактивен речник | Клучните поими се дефинирани на јасен македонски јазик |
| 💡 Совет | Советот го дава „рамката" за одговорот — ученикот само ја пополнува |
| MCQ прашања | Не бараат јазична продукција — само препознавање |
| Повратна информација со клучни зборови | „Ти кажа `помош`, додај уште еден детал" — го учи јазикот на одговарање |

#### Постепен пристап кај јазични тешкотии

```
Фаза 1: Само MCQ прашања (прв контакт со текстот)
Фаза 2: Отворени прашања со активен Совет
Фаза 3: Отворени прашања без Совет (кога ученикот ќе стекне самодоверба)
```

### 7.6 За ученици со аутизам / нарушувања на сензорната обработка

Предвидливоста и јасната структура се клучни. Неочекуваните звуци, анимации или промени во рутина можат да предизвикаат дискомфор.

#### Поставување пред часот — аутизам

1. **Исклучете го звукот на уредот** ако ученикот е чувствителен на звуци (🔊 функцијата нема да се активира случајно)
2. **Покажете ја структурата однапред**: „Прво читаме, потоа 12 прашања, потоа БРАВО — тоа е сè"
3. **Доделете иста приказна секој пат** при прво запознавање — познатото го намалува стресот
4. **Исклучете го тajмерот** ако временскиот притисок предизвикува анксиозност

#### Предности на апликацијата за ученици со аутизам

- **Линеарна, предвидлива структура**: секоја приказна е исто организирана
- **Нема изненадувачки pop-up прозорци**: сè се активира само по клик
- **Конзистентен интерфејс**: ист изглед за секоја приказна
- **Работа во тишина**: целата работа е можна без звук

### 7.7 Упатство за наставниците — Работа со мешано одделение

#### Диференцирање во еден час — три нивоа

Апликацијата не бара различни материјали за различни ученици. **Еден текст, три нивоа на поддршка:**

| Ниво | Ученик | Поставување во апликацијата |
| ---- | ------ | ---------------------------- |
| **Поддршка** | Ученици со ПОП, слаби читачи | `A+` фонт + 💡 Совет секогаш активен + само MCQ во прва рунда |
| **Стандардно** | Просечен ученик | Стандардни поставувања, Совет по потреба |
| **Напредно** | Силни читачи | Без Совет, тajмер активен, и двата типа прашања |

#### Следење на ученици со ПОП — знаци на таблата

Наставникот ги гледа сите ученици истовремено. За ученик со ПОП, обрнете внимание на:

| Знак на таблата | Можна причина | Акција |
| --------------- | ------------- | ------ |
| Напредок 0% по 10 мин. | Тешкотии со читање | Приодете лично, проверете дали разбира |
| Стагнација на исто прашање >5 мин. | Потешко прашање | Активирајте Совет за тој ученик |
| Многу брзо завршување (< 5 мин.) | Импулсивно одговарање | Проверете ги одговорите — MCQ со многу погрешни |
| Не се приклучил на сесијата | Технички проблем | Проверете го уредот |

#### Белешка за документацијата на ИОП

Одговорите зачувани во Firebase можат да послужат како **документација за напредокот** на ученикот со ИОП:

- Датум и час на работа
- Тип на текст (наративен/информативен)
- Точност на MCQ одговорите
- Содржина на отворените одговори

> Извезете ги податоците од таблата за да ги зачувате пред затворање на сесијата.

## 8. ЧПП

**Прашање: Зошто другите приказни се сиви и не можам да ги отворам?**
→ Наставникот доделил одредена приказна за денешниот час. Само таа може да се отвора. По завршувањето, ако наставникот ја отклучи сесијата, ќе можете да работите на другите.

---

**Прашање: Кодот на наставникот не работи.**
→ Проверете дали кодот е точно препишан (6 знаци). Не е важно мали/големи букви. Ако кодот сè уште не работи, наставникот треба да ја провери интернет врската на неговиот уред и да ви го даде кодот повторно.

---

**Прашање: Изгубив напредокот / медалките.**
→ Медалките се зачувуваат во прелистувачот (localStorage). Ако го избришавте кешот на прелистувачот или работите на нов уред, медалките нема да бидат видливи. Одговорите во сесијата со наставник се зачувани на сервер и не се губат.

---

**Прашање: Апликацијата не се вчитува добро на мобилен.**
→ Обидете се да го ротирате телефонот хоризонтално (landscape). Ако проблемот продолжи, рестартирајте го прелистувачот.

---

**Прашање: Не ги слушам прашањата на глас (нема копче 🔊).**
→ Копчето 🔊 се прикажува само ако вашиот уред поддржува македонски глас. На Windows 10/11 со инсталиран македонски јазичен пакет ова работи. На мобилни уреди резултатите варираат.

---

**Прашање: Дали апликацијата собира лични податоци?**
→ Учениците се идентификуваат само со прекарче — нема e-пошта, нема лозинка, нема лични информации. Наставниците се логираат само со PIN. Апликацијата е усогласена со ЗЗЛП (македонски GDPR).

---

**Прашање: Каде можам да ги видам одговорите на ученикот откако ќе заврши часот?**
→ Во таблата за наставници, кликнете на ime на ученикот за детален преглед на сите одговори со временски марки.

---

**Прашање: Можам ли да ја користам апликацијата на домашен компјутер со деца?**
→ Да. Одете на линкот → директно изберете приказна → без регистрација. Родителите не треба PIN.
