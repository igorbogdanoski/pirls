# Open-Response Rubric Framework (PIRLS-aligned)

## Goal
Standardize scoring of text responses with transparent criteria for teachers and reviewers.

## 4-Level Rubric
- Level 0 (0 pts): Off-topic, empty, or unintelligible
- Level 1 (1 pt): Partial response, one relevant idea, weak textual grounding
- Level 2 (2 pts): Relevant response with two supporting elements from text
- Level 3 (3 pts): Complete reasoning, clear textual evidence, coherent explanation

## Scoring Dimensions
- Relevance to question intent
- Evidence from text
- Completeness of reasoning
- Language clarity

## Mapping to Existing Criteria
Current app checks keywords and length. Recommended mapping:
- 0 matched keywords -> Level 0
- 1 matched keyword + minimum length -> Level 1
- 2 matched keywords + minimum length -> Level 2
- 3+ matched keywords + explicit evidence phrase -> Level 3

## Teacher-facing Feedback Phrases
- Level 0: "Read the paragraph again and identify one key idea."
- Level 1: "Good start. Add one more detail from the text."
- Level 2: "Strong answer. Add a direct textual reference for full credit."
- Level 3: "Excellent evidence-based explanation."

## Calibration Protocol
- Select 20 student responses per story
- Two reviewers score independently
- Calculate agreement rate
- Resolve disagreements and update examples

## Evidence Anchors (per question)
For each open question, maintain:
- Expected key concepts (keywords)
- Acceptable paraphrases
- Example responses for Levels 0-3

## Review Artifact
During external review, provide:
- Rubric definitions
- Example scored scripts
- Inter-rater agreement summary
